# Glossary
- SER: Speech Emotion Recognition - The process of determining the emotion/tone behind a given phrase or statement.
- CUDA: Compute Unified Device Architecture - NVIDIA's parallel GPU compute API, used to perform computations on the GPU.
- HuBERT: Hidden-Unit BERT - Speech model which learns patterns from speech.
- ONNX: Open Neural Network Exchange - A file format and runtime which allows deep learning models to be saved and loaded from a unified file format.
